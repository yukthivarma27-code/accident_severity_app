## 📖 Overview

The **Accident Severity Prediction System** is a machine learning application developed to estimate the severity of road accidents based on multiple influencing factors, including road conditions, environmental parameters, traffic attributes, and vehicle-related information.

By leveraging predictive analytics, the system assists in identifying high-risk scenarios, enabling proactive safety measures, better traffic planning, and informed decision-making for transportation stakeholders.

## 🚀 Getting Started

### 1️⃣ Clone Repository

```bash
git clone https://github.com/yukthivarma27-code/accident_severity_app.git
cd accident_severity_app
```

### 2️⃣ Create Virtual Environment

```bash
python -m venv venv
```

Activate environment:

**Windows**

```bash
venv\Scripts\activate
```

**Linux / macOS**

```bash
source venv/bin/activate
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run Application

```bash
python app.py
```
